توزيع الفريق ومسار العمل الخمسة

تم تخفيف الحمل عن إسراء بنقل جزء من الواجهات (ويب + أندرويد) إلى حنين، مع بقاء صبري كما هو. أدناه: مسؤوليات إجمالية ثم كل مرحلة مع الملفات.

آخر تحديث المستودع (أبريل 2026): التعديلات المدمجة تشمل فرع rewrite/sabry-web-auth على github.com/SabryAlaa10/mawid-plus. أبرز ما دخل الكود: هجرات 016–020 (قيود تحديث إعدادات الطابور، إلغاء موعد المريض عبر RPC، بذرة عيادة، تقييم طبيب، نشر Realtime لجدول الأطباء)، أيقونة التشغيل Adaptive (خلفية #1A73E8 + حقيبة إسعاف بيضاء وعلامة +)، MawidRegion.kt وregion.js، تشديد supabaseClient مع bootstrap.jsx وauthErrorMessage.js، وتحديثات واجهات/منطق الحجز والمواعيد والطابور والملف والبحث والإشعارات، وإصدار التطبيق 1.0.6.

مساعد RAG (المرحلة ٥): ما زال مخططاً — ملفات مثل RAGRepository.kt غير موجودة بعد؛ الهجرات 016–020 لا تحل محل pgvector/الـ Edge Function للمساعد.
