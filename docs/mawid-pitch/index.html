منصة موعد+ — ربط المريض بالعيادة والطبيب

نظام متكامل لإدارة الحجز، الطابور، والملفات؛ مع لوحة ويب للأطباء وتطبيق أندرويد للمرضى، ومساعد ذكي للإرشاد الصحي العام.

An integrated stack for bookings, live queues, and profiles: a React web dashboard for doctors, an Android patient app, and an AI assistant for general health guidance.

Supabase + RLS React · Vite · Tailwind Kotlin · Jetpack Compose FastAPI · Mawid Assistant

الويب — بوابة الطبيب

Web — Physician portal

تسجيل دخول آمن، لوحة تحكم، جدول المواعيد الحي، إدارة طابور الانتظار، الجدول والرسوم، والملف الشخصي الظاهر للمرضى.

التطبيق — تجربة المريض

Mobile — Patient experience

تسجيل بالهاتف أو Google، الرئيسية مع الموعد القادم، البحث عن الأطباء، مواعيدي، الحساب، ومساعد موعد الذكي.

الخلفية والذكاء

Backend & intelligence

قاعدة بيانات موحّدة عبر Supabase، وواجهة برمجة للمساعد الطبي تدعم المحادثة واقتراح التخصصات والمواعيد ضمن سياسات المنصة.
