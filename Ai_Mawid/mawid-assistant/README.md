# Mawid Assistant

مساعد طبي ذكي يساعد المرضى في العثور على الأطباء المناسبين وحجز المواعيد باستخدام الذكاء الاصطناعي.

## الميزات
- محادثة ذكية مع الذكاء الاصطناعي
- كشف التخصص الطبي من الأعراض
- حجز مواعيد مع الأطباء
- قاعدة معرفة طبية

## التثبيت
1. قم بتثبيت المتطلبات: `pip install -r requirements.txt`
2. أضف متغيرات البيئة في ملف `.env`
3. شغل الخادم: `uvicorn app.main:app --reload`

## الاستخدام
- API documentation: http://localhost:8000/docs
